# 🧮 Casio FX-82MS Scientific Calculator

A web-based scientific calculator inspired by the classic **Casio FX-82MS**, built with pure HTML, CSS, and JavaScript — no frameworks, no APIs.

## 🌐 Live Demo
[Click here to open the calculator](https://anant-083.github.io/Scientific-calculator/)

## ✨ Features
- Classic Casio FX-82MS inspired design
- Basic arithmetic: addition, subtraction, multiplication, division
- Scientific functions: sin, cos, tan (and inverses)
- Logarithms: log, ln
- Powers and roots: x², √, xʸ
- Constants: π, e
- Percentage calculations
- DEG / RAD mode toggle
- ANS key (reuse last result)
- Expression history (▲ key)
- Cursor navigation (◀ ▶)
- Keyboard support
- Fully responsive

## 🛠️ Built With
- HTML5
- CSS3
- Vanilla JavaScript
